Help & Documentation

Welcome to the official documentation for the BIMetic Saved Viewpoints Manager for Autodesk Navisworks. This guide covers installation, usage, and uninstallation.

1. Installation and Uninstallation

How to Install

  1. Download the `.msi` installer package from the Autodesk App Store.
  2. Make sure Autodesk Navisworks is completely closed.
  3. Double-click the installer and follow the on-screen prompts.
  4. Launch Navisworks. The BIMetic tool will appear in the "Add-ins" or "Tool Add-ins" ribbon tab.

How to Uninstall

  1. Close Autodesk Navisworks.
  2. Open the Windows Control Panel and navigate to Programs and Features.
  3. Locate "BIMetic Saved Viewpoints Manager" in the list of installed programs.
  4. Click Uninstall and follow the prompts to safely remove the plugin.

2. General Usage Instructions

Click the "BIMetic Manager" icon in your Navisworks ribbon to open the Cloud-App interface. From there, you can utilize the following features:

Bulk Formatting & Auto-Counting

Select the folders you wish to process from the tree view. You can choose a starting ID (e.g., 001) and apply multiple path formats. The engine can intelligently append the total view count to folder titles recursively (e.g., "Architecture (42)").

Backup & Restore Snapshot Engine

Before executing any bulk formatting, the plugin automatically creates a local JSON snapshot of your entire viewpoint tree. If you make a mistake, simply navigate to the Restore tab, select the latest snapshot timestamp, and instantly revert your changes without losing clash data.

Duplicate Detection BETA

The Duplicate Detector scans your selected folders and mathematically analyzes the 3D Euclidean distances between camera target positions, quaternion rotations, and markup counts. It intelligently classifies viewpoints into three categories:

  • TRUE_DUPLICATE: Identical camera coordinates and markup data. Recommended for cleanup.
  • INTENTIONAL_SEQUENCE: Similar target but different camera distance/angle. Safe to keep.
  • REUSED_VIEWPOINT: Clean, non-duplicate views.

⚠️ Warning: Experimental Feature
The Duplicate Detector is currently in Beta. Because it relies on mathematical heuristics, it may occasionally flag intentional views as duplicates. Always use the built-in Backup & Restore snapshot engine before running a bulk cleanup operation.
