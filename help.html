Help & Documentation

Welcome to the official documentation for the BIMetic Saved Viewpoints Manager for Autodesk Navisworks. This guide covers installation, authentication, and usage of the core pipeline features.

1. Installation and Authentication

How to Install

  1. Download the `.bundle` or `.msi` installer package from the Autodesk App Store.
  2. Make sure Autodesk Navisworks is completely closed.
  3. Run the installer and follow the on-screen prompts.
  4. Launch Navisworks. The BIMetic tool will appear in the "Add-ins" or "Tool Add-ins" ribbon tab.

Autodesk Authentication

Our plugin is secured using the official Autodesk App Store Entitlement API. When you click the Ribbon button, the C# engine checks your current logged-in Autodesk ID against the App Store.

Note: You must be signed into Navisworks with the exact Autodesk ID used to purchase the subscription, or you will receive a "License Required" error.

2. Pipeline Formatter

The core feature of BIMetic is the bulk formatting pipeline. You can access this via the "Pipeline Formatter" tab.

Target Scope Selection

The UI reads your live Navisworks folder structure. Use the checkboxes to select which folders to process. Tip: Shift+Click a parent folder to instantly toggle all of its subfolders.

Configuration Rules

  • Append Current Date: Appends today's date to the viewpoint name.
  • Prefix Folder Number: Pre-pends the index of the viewpoint within its folder (e.g., "01 - ViewName").
  • Dynamic Auto-Counting: Automatically counts how many viewpoints are inside a folder and appends the total to the folder name itself (e.g., "Architecture (42)").
  • Character Mapping Matrix: Add custom find-and-replace rules to sanitize your data before formatting (e.g., replace "_" with " ").

3. Markup Tracker

Go to the "Viewpoint Markups" tab to analyze your model for active redlines.

  • The engine scans every viewpoint in your scope for hidden Redline data.
  • It presents a summary list of all views containing markups.
  • Click the Target Reticle Icon next to any flagged viewpoint to instantly activate that view in Navisworks and see the markup!

4. Duplicate Detector

The Duplicate Detector scans your selected folders and analyzes the viewpoints to identify exact matches.

  • It compares camera target positions, angles, and names.
  • Any viewpoints that are identical in these properties are flagged as True Duplicates.
  • This allows you to safely clean up redundant views that were accidentally copied or created.

5. Auto-Backup & Restore

We built an infallible safety net into the plugin. Before any operation executes, the C# engine silently captures an XML snapshot of your entire Saved Viewpoints tree.

  • If you make a mistake, click the Restore Pipeline tab.
  • Select the exact timestamp of the backup.
  • Choose whether to restore just Viewpoint Names, or the full Folder structure.
  • The plugin will instantly roll back the Navisworks tree.
